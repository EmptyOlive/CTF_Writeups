# Enigma

# Level 1
The Enigma was a cipher machine used in the second world war. The German military used it to send secret messages. They believed these messages could not be understood by anyone else.

# Level 2
Enigma machines encrypted every letter a person typed and could generate a cipher using one of nearly 159 quintillion possible settings.

# Level 3
British mathematicians Alan Turing and Gordon Welchman invented a machine that could solve the enigma settings. This helped the allies decrypt the messages sent by the Germans.

# TypeX

# Level 1
The TypeX machine used similar technology to the enigma and was developed in Britain in 1934. The Australian government used it throughout the second world war. 

# Level 2
The allies used the typex to protect their own top-secret communications. It was more secure than the German enigma. 

# Level 3
Several Versions of the Typex machine were developed, Each with a different and particular purpose. It was an adaptable machine so it could easily be modified.

# Code Book

# Level 1
A code book is a document that allows the user to encode or decode a message. To decode a message, both the sender and the receiver must have identical code books.

# Level 2 
The Australian Army used code books to send updates from the battlefield and to receive orders. These books were changed regularly to make it more difficult to decrypt a message if it was captured by the enemy.
