Firstly I tried to see if there was any message from the combination of flags themselves.

No message was able to be created. I then swap to potential capital cities.

Washington DC, Honiara, Islamabad, Canberra, Hanoi, Nairobi, Addis Ababa, Tbilisi, Islamabad, Ottawa, N’Djamena, Suva
Funafuti, Lima, Ankara, Gaborone, Islamabad, Santiago, Astana, Georgetown, Oslo, Lilongwe, Dublin, Manila, Ouagadougou, Oranjestad, New Delhi, Ottawa, Nouakchott, Buenos Aries, Ljubljana, Ulaanbaatar, Edinburgh

Using the first letter from these we get the following message:
```
Which nations flag is a goldmoon on blue
```
***Answer: Palau***
